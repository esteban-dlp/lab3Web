# La Brújula de Ceniza

Historia interactiva hecha únicamente con HTML.

## Estructura
- `index.html` es el inicio
- `house/` contiene escenas de la casa
- `forest/` contiene escenas del bosque
- `endings/` contiene los finales
- `images/` contiene las imágenes

## Requisitos cumplidos
- Solo HTML
- Sin CSS
- Sin JavaScript
- Varias páginas HTML
- Links relativos y absolutos
- Breadcrumbs
- Uso de etiquetas semánticas
- Al menos una imagen
- Varios finales
- Loop de navegación
